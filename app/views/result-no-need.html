You may still choose to send a return

Even if you do not have to, you may want to send a return if you:

  • want to claim tax relief on pension contributions, charitable donations or work expenses
  • want to prove you are self-employed, for example to claim tax-free childcare or Maternity Allowance
  • are not resident in the UK for tax purposes

If you previously sent a return

If HMRC has sent you a notice to complete a Self Assessment tax return, you must complete it even if you do not owe any tax. Contact HMRC to ask them to cancel the return if you believe you no longer need to file one.
